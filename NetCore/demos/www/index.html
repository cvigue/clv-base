🚀 Simple Config-Driven Server

Welcome!

This is a simple HTTPS server built with ClvLib components:

  • ✅ Config system for JSON configuration
  • ✅ HttpCore for HTTP/HTTPS protocol handling
  • ✅ Static file serving
  • ✅ API endpoint routing
  • ✅ SSL/TLS support

Available API Endpoints

  • GET /api/hello - Test endpoint
  • GET /api/config - Server configuration info

How It Works

The server loads configuration from server_config.json and serves:

  1. API Routes: Handled first by ApiRouter (/api/*)
  2. Static Files: Fallback to HttpStaticRouter for files in www/
  3. 404s: Automatic for unhandled routes

Built with C++20 coroutines, ASIO, and OpenSSL
