HttpCore Test Server

Welcome!

This is a test server built with the HttpCore library.

Available Endpoints:

  • GET / - This page
  • GET /test.txt - Test text file
  • GET /api/hello - Simple API greeting
  • GET /api/json - JSON response example
  • POST /api/echo - Echo back the request body
  • GET /api/headers - Show request headers

Test Commands:

# Basic GET request
curl -k https://localhost:8443/

# Get JSON
curl -k https://localhost:8443/api/json

# Echo POST data
curl -k -X POST -d "Hello World" https://localhost:8443/api/echo

# View headers
curl -k https://localhost:8443/api/headers
